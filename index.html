JavaScript Timer Functions

1. setTimeout

Definition: Executes a function after a specified delay.

Purpose: To delay the execution of a function.

Applications: Delayed actions, animations, etc.

2. clearTimeout

Definition: Clears a timeout set with setTimeout.

Purpose: To stop the execution of the function set to run after the delay.

Applications: Cancel delayed actions if conditions change.

3. setInterval

Definition: Executes a function repeatedly, with a fixed time delay between each call.

Purpose: To run a function repeatedly at set intervals.

Applications: Repeating animations, updating the UI at regular intervals, etc.

4. clearInterval

Definition: Clears a timer set with setInterval.

Purpose: To stop the execution of the interval function.

Applications: Stopping repeating actions when conditions change.

JavaScript Date Object Exercises

1. Create a Date Object:

Write a script to create a Date object for the current date and time.

2. Display the Current Year:

Use a getter method to display the current year.

3. Format the Date:

Use toLocaleDateString() to display today's date in the format mm/dd/yyyy.

4. Find the Day of the Week:

Determine the day of the week (e.g., Monday, Tuesday) for today's date.

5. Calculate Age:

Given a birthdate (e.g., 1995-12-17), calculate the current age.

Answer Key (Pseudocode):

  1. Create a Date Object:

    SET currentDate TO new instance of Date

    DISPLAY currentDate

  2. Display the Current Year:

    SET currentYear TO year from currentDate (using getFullYear method)

    DISPLAY currentYear

  3. Format the Date:

    SET formattedDate TO currentDate formatted as "mm/dd/yyyy" (using toLocaleDateString method)

    DISPLAY formattedDate

  4. Find the Day of the Week:

    SET daysArray TO array of days ["Sunday", "Monday", ..., "Saturday"]

    SET dayOfWeek TO day from currentDate (using getDay method) indexed in daysArray

    DISPLAY dayOfWeek

  5. Calculate Age:

    SET birthDate TO new Date instance with given birthdate string

    SET ageDifference TO difference in milliseconds between currentDate and birthDate

    SET age TO convert ageDifference into years

    DISPLAY age
