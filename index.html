Module 02 Lesson 04

Functions

  1. Functions are like machines that perform recipes, guiding step by step to automate tasks.
  2. Automate repetitive processes to avoid rewriting the same instructions.
  3. Use functions to perform tasks, like greeting different people by name.

Parameters

  1. Parameters are the ingredients needed for the function's recipe.
  2. They are bits of information to customize the function's task.
  3. In a greeting function, the person's name is a parameter indicating who to greet.

Return Statement

  1. The return statement is the end product, like the cake from a recipe.
  2. It provides the resolt of the function's computation or action.
  3. For a sum calcolation, it gives you the total sum.

Default Parameters

  1. Default parameters are like backup ingredients in your pantry.
  2. Used when no specific value is provided for a parameter.
  3. Ensure the function can run with a default value, like a default greeting message.

In summary, JavaScript functions automate tasks with instructions for processing data (parameters), yielding outcomes (return statement), and ensuring operability with defaults (default parameters).
